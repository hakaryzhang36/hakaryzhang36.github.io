博客网站的目标？

这篇文章的基本目标：

  1. 基于 Github Page 搭建个人博客
  2. 可以将 Markdown 直接转换成一篇博客
  3. 将博客源码、网站资源分别部署到不同的 Github 仓库

前置环境

本篇文章的博客是在 Windows 中操作的，如果你想了解其他系统中的环境搭建，可以参考hexo 官方文档（简单明了）。

  1. 安装 Node.js
  2. 安装 hexo

开始建站 hexo

基本思路

hexo 作为 nodejs 的一个库，基本功能就是实现了将不同形式的文章书写语言转换为一个静态页面，然后再自动生成网站相应的资源。

所以，我们要做的大致分为以下几步：

  1. 初始化，在一个目录初始化 hexo 项目，该目录下会有 hexo 配置文件、主题、网站资源等内容。
  2. 写作，写文章
  3. 部署，将生成的网站资源部署到 Github 仓库
  4. 访问，通过 Github 仓库的 Github Page 访问我们的博客页面。
